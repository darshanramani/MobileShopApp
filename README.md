# TECHIFY 📱

Modern mobile shopping application developed using Kotlin and Jetpack Compose in Android Studio.

---

## 🚀 Features

* Splash / Welcome Screen
* 2 Column Product Grid
* Product Images
* Product Ratings ⭐
* Favorite Products ❤️
* Add to Cart System
* Quantity Update (+ / -)
* Shopping Cart Dialog
* Dynamic Total Calculation
* Checkout Popup
* Dark Mode / Light Mode 🌙
* Modern UI Design

---

## 🛠️ Technologies Used

* Kotlin
* Android Studio
* Jetpack Compose
* Material 3

---

## 📦 Products Included

* AirPods
* Gaming Headset
* Laptop
* Speaker
* Smart Watch
* Smart TV

---

## ▶️ How to Run

1. Open project in Android Studio
2. Wait for Gradle Sync
3. Connect Emulator or Android Phone
4. Click ▶ Run button
5. App will launch automatically
